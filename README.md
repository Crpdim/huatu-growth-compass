# 职图有声 Demo

本项目用于「2026 AI 先锋未来人才大赛」华图教育命题的方案设计与 Demo 开发。

- [比赛官网](https://activity.feishu.cn/future-talent#challenge)
- [华图教育命题](https://activity.feishu.cn/future-talent?detail=huatujiaoyu)

## 产品定义

> 职图有声是一套 AI 驱动的大学生成长实验系统，通过“证据画像—多路径推演—成长实验—动态校准”，把学生对未来的模糊焦虑拆成可验证的问题，帮助他们用真实行动逐步找到方向。

产品持续帮助用户回答四个问题：我是谁、我现在在哪、我能去哪、下一步怎么走。

核心闭环：

```text
定位 → 定向 → 导航 → 校准
画像 → 推演 → 实验 → 新证据 → 再校准
```

## 当前 Demo 的唯一目标

用一个虚构大二计算机专业学生“林小北”跑通当前重点 Demo：

> 产品宣传首页 → 人生模拟器 → 模范公务员结局 → 补全任务画像 → 八维雷达画像 → 三条路径与同路人案例 → 导入成长管理 → Agent 会话与独立任务管理中的执行、查询和动态校准

Demo 首页升级为以产品视频为中心的宣传页，“开始体验”链接进入“模拟人生”小游戏。用户可以从小游戏右上角“没有人生目标？”直接进入原有方向探索，也可以完成体制内人生并达成“模范公务员”，再通过“补全任务画像”进入真实信息补充。小游戏当前完整演示体制内人生，其余人生路线保留为静态入口。

产品逻辑支持考研、公考/考编和企业求职三条路径，比赛 Demo 使用考公完整演示一段阶段性规划，另外两条作为对照。八维画像将性格适配、专业能力、兴趣匹配度、学习与知识、抗压与情绪、沟通协作、健康情况和家庭情况统一放进八轴雷达；界面只展示面向用户的结论、依据和置信度，不展示内部信号、权重公式或提示词。成长管理以 Agent 会话为主入口，并保留一个与对话共享状态的独立任务管理页；学生可以直接新增、编辑、暂停或删除任务，也可以从单个任务旁打开带快捷建议和自由输入框的 AI 对话助手，演示任务拆解和官方网址检索。周回顾和阶段总结继续在会话内呈现。AI 服务异常时，静态内容仍能支撑完整演示。

## 运行 Demo

第一版可点击原型位于 [`demo/`](./demo/)，使用固定样例和静态 fallback 跑通完整闭环。

```bash
cd demo
npm ci
npm run dev
```

当前版本不接真实 AI、后端、登录或实时外部数据，适合用于产品走查和比赛演示。
